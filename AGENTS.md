## Project Rules

This repository is a small Python learning and environment test project.

## Structure

- `README.md`: project overview, setup, and run instructions.
- `pythonProject/DeepLearningProjrct/`: Python scripts for PyTorch and NumPy practice.

## Naming

- Use English file and directory names for code.
- Prefer clear script names such as `train_model.py`.
- Keep documentation in Markdown files at the repository root unless the project grows.

## Validation

After changing Python code, run:

```powershell
python pythonProject\DeepLearningProjrct\train_model.py
```

After changing Markdown only, review the rendered Markdown or inspect the file content.
